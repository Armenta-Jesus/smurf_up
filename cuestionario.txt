5. Describe los protocolos que existen en internet, en específico HTTP y HTML.
R= El protocolo HTTP (HyperText Transfer Protocol) funciona de manera que cuándo tú tecleas una dirección en tu navegardor de preferencia y pulsas enter, en ese momento tu computadora se comunica con el servidor y el servidor le responde con lenguaje HTTP y se termina encontrando la dirección que estás buscando. El protocolo HTML (Hyper Text Markup Languaje) es el lenguaje que se utiliza para darle un formato a una página web (tamaño de letra, negritas, título, etc).
